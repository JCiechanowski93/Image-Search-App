## General info

Simple web application whose purpose is to search for images that are downloaded from Unsplash Image API.

I did it to better understand React.

## Technologies

- React v16.3.0
- Axios
- Webpack
- Grid CSS
- Semantic UI

## Setup

To run the project:
```
 1. npm i
 2. $ npm run start
 3. Open  local host http://localhost:8080
```
